# Pomodoro Temporizer!⏰
EN:
  The *pomodoro technique* is a time management method developed by Francesco Cirillo in the late 1980s.
The technique uses a timer to break down work into intervals, usually **25 minutes in length**, separated by short breaks.
                              Its purpose is to **improve focus and productivity**

Tutorial:
Execute the **main.py** file *(make sure you have the proper modules installed)*, then click on the "?" icon for further **usage instructions**. You can change the language in configurations.

ES:
  La *técnica pomodoro* es un método de gestión del tiempo desarrollado por Francesco Cirillo a finales de la década de 1980.  
Esta técnica utiliza un temporizador para dividir el trabajo en intervalos, normalmente de **25 minutos de duración**, separados por **descansos cortos**.
                              Su objetivo es mejorar el **enfoque y la productividad.**

Tutorial:
Ejecuta el archivo main.py (asegúrate de tener los módulos adecuados instalados), luego haz clic en el icono "?" para obtener **más instrucciones de uso**. Puedes cambiar el idioma en configuraciones.
